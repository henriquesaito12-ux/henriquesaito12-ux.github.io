Design System Trackage

Construcao de um design system do zero para unificar a linguagem visual e acelerar a entrega de produto.

Empresa

Trackage

Meu papel

Design System Lead

Ferramentas

Figma, Storybook, Tokens Studio

Imagem — Visao geral do design system

Contexto

A Trackage tinha multiplos produtos com interfaces desalinhadas. Cada squad criava componentes do zero, gerando retrabalho e inconsistencia na experiencia do usuario.

Problema

Sem uma fonte de verdade para componentes e padroes, o time perdia tempo recriando elementos e discutindo decisoes que ja haviam sido tomadas. A velocidade de entrega estava comprometida.

Imagem — Inventario de componentes existentes
Imagem — Tokens de cor e tipografia

Solucao

Construi o design system em camadas: tokens de design (cores, espacamento, tipografia), componentes atomicos, e padroes compostos. Tudo documentado e versionado no Figma com espelho no Storybook.

Criei um processo de governanca com rituais de review e contribuicao para que qualquer designer ou dev pudesse propor novos componentes.

Imagem — Biblioteca de componentes no Figma

Impacto

O design system foi adotado por todas as squads. Reducao no tempo de design e desenvolvimento de novas features. Consistencia visual entre todos os produtos da empresa.
