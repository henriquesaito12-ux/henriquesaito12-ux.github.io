Design System Trackage

Design System Trackage UX/Product Designer 2022

Construção de um design system do zero para unificar a linguagem visual e acelerar a entrega de produto.

01 VISÃO GERAL

Empresa

Trackage

Meu papel

Design System Lead

Ferramentas

Figma

A Trackage tinha múltiplos produtos com interfaces desalinhadas. Cada squad criava componentes do zero, gerando retrabalho e inconsistência na experiência do usuário. Construí o design system de ponta a ponta para unificar a linguagem visual e acelerar a entrega de produto.

02 PROBLEMA

Sem uma fonte de verdade para componentes e padrões, o time perdia tempo recriando elementos e discutindo decisões que já haviam sido tomadas. A velocidade de entrega estava comprometida. Interfaces inconsistentes entre produtos geravam confusão nos usuários e dificultavam a manutenção.

03 SOLUÇÃO

Construí o design system em camadas: tokens de design (cores, espaçamento, tipografia), componentes atômicos e padrões compostos. Tudo documentado e versionado no Figma. Criei um processo de governança com rituais de review e contribuição para que qualquer designer ou dev pudesse propor novos componentes. O sistema foi pensado para escalar com a empresa, não apenas resolver o problema imediato.

04 RESULTADO

Resultado

O design system foi adotado por todas as squads. Redução no tempo de design e desenvolvimento de novas features. Consistência visual entre todos os produtos da empresa.

Meu papel

Liderei a construção do design system de ponta a ponta: inventário de componentes, definição de tokens, criação da biblioteca no Figma e processo de governança. Atuei como ponte entre design e engenharia para garantir que o sistema fosse adotado de forma prática, não apenas teórica.
