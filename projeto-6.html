Exploracao de Dashboard com foco em interacao

Design de dashboards interativos que permitem explorar dados de forma intuitiva, sem depender de relatorios estaticos.

Tipo

Data Visualization / UI

Meu papel

UX/UI Designer

Ferramentas

Figma, D3.js references, Notion

Imagem — Dashboard principal

Contexto

A equipe de gestao dependia de relatorios em PDF e planilhas para acompanhar metricas. Os dados estavam dispersos e a visualizacao nao facilitava a tomada de decisao.

Problema

Dashboards existentes eram densos e estaticos. Os usuarios nao conseguiam explorar os dados por conta propria — dependiam de analistas para gerar cada corte ou visualizacao diferente.

Imagem — Pesquisa com usuarios gestores
Imagem — Wireframes de exploracão

Solucao

Projetei dashboards com filtros dinamicos, drill-down em graficos e tooltips contextuais. O foco foi permitir que o usuario explorasse os dados de forma autonoma, sem conhecimento tecnico.

Cada visualizacao foi pensada para contar uma historia: do panorama geral ao detalhe, com transicoes suaves entre niveis de informacao.

Imagem — Interacao de drill-down

Impacto

A equipe de gestao passou a tomar decisoes baseadas em dados de forma autonoma. Reducao na dependencia de analistas para relatorios ad-hoc e aumento na frequencia de uso dos dashboards.
